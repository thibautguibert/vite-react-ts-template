## Available commands

Run in development mode

```bash
  yarn dev
```

Create production build

```bash
  yarn build
```

Run ESLint linting

```bash
  yarn lint
```

Run Prettier formatting

```bash
  yarn format
```

Run TypeScript compiling and check for erros

```bash
  yarn tsc-lint
```

Serve production build locally

```bash
  yarn preview
```

Run your tests

```bash
  yarn test
```
